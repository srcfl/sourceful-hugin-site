Connect your local agent.

A Hugin agent runs on your network — your laptop, a Pi, or an existing forty-two-watts box — and gives the web app real LAN access: ARP scanning, Modbus probes, live Lua driver testing.

It's optional. Browsing the catalog and authoring drivers via AI works without one. Pairing unlocks the on-the-wire side.

Pair an agent

Default for Hugin Agent is http://localhost:19090. Forty-two-watts boxes typically run on http://<host>:8080.

The agent prints this on first launch. Stays in your browser — never sent to Sourceful's servers.

How to get an agent
Option 1

Hugin Agent

Lightweight Go binary. Runs on your laptop or a Raspberry Pi. ~5 MB, no dependencies, single static binary. Prints a pairing token on first start.

github.com/srcfl/hugin-agent →
Option 2

forty-two-watts

Already running 42W on a home server? It speaks the same agent protocol. Point Hugin at http://<host>:8080 and reuse the existing token. One stack, two surfaces.

github.com/frahlg/forty-two-watts →

Both are open source. Sourceful never sees your LAN traffic — the agent talks to your browser directly over http://localhost or your private network. Tokens are stored only in this browser's localStorage; clearing the pairing wipes them.
