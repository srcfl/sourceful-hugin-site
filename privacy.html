Privacy.

What we collect, what we don't, and what you can do about it. Short, plain English, GDPR-aware. If a section is too vague to act on, tell us — we'll rewrite it.

Last updated 2026-05-12.

What we collect

  • GitHub identity — your login and numeric user_id, from the OAuth sign-in. Used to authenticate workbench requests, attribute driver PRs, and scope NATS credentials. Kept while your account is paired.
  • AI usage — one structured log line per call to /v1/ai/messages: your login, the model name, input/output token counts, status code, response size, timestamp. Used to size the free tier and spot abuse. Aggregated for capacity planning. Never sold.
  • Agent metadata — when you pair a remote agent we mint a unique agent_id and link it to your GitHub user ID. Held in memory; lost when the API restarts.
  • Server access logs — standard request metadata (path, status, latency, no body) for ~7 days, used for debugging.

What we don't collect

  • Your email, real name, or any contact info beyond what GitHub already makes public.
  • Anything the agent sees on your network. Modbus reads, ARP scans, Lua emissions — none of it leaves your browser.
  • Driver source while you're drafting. The AI proxy forwards your prompts to Anthropic; we log token counts, not content. The driver becomes public only when you click Submit as PR and the commit lands in srcfl/hugin-drivers.
  • IP geolocation, device fingerprints, browser-side analytics. No cookies — auth lives in localStorage, scoped to the workbench origin.

Third parties we use

  • GitHub — OAuth + repository hosting. Their privacy statement applies to your GitHub account.
  • Anthropic — AI inference. Your prompts are sent to Anthropic when you use AI features. Their privacy policy applies. Sourceful holds the API key; you don't sign in to Anthropic.
  • AWS (Frankfurt, eu-central-1) — runs our API and NATS broker. Data residency is EU.
  • Cloudflare Workers — serves the workbench HTML. Cloudflare sees your IP for the duration of each page load.

Your rights (GDPR)

  • Access — ask us what we hold on you. Email legal@sourceful.energy.
  • Delete — revoke the GitHub OAuth scope at github.com/settings/applications to disconnect Hugin. Usage logs are wiped within 30 days. PR commits you authored remain on GitHub (their property + your authorship).
  • Portability — drivers you authored are MIT-licensed; export at will.
  • Object — stop using the workbench. The agent itself collects nothing; uninstall it (brew uninstall hugin-agent or delete the binary) and you're done.

Local storage

The workbench keeps a few things in your browser's localStorage:

  • hugin.gh_token — your GitHub OAuth token, used by the workbench to call the API.
  • hugin.agent_* — pairing details for the agent you're connected to (URL + token for local, agent_id + NATS creds for remote).
  • hugin.drafts — driver drafts you're working on. Never sent anywhere.

These never leave your browser. Clear them at /settings.html (Clear pairing) or in browser DevTools.
