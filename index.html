Install

One command.

The agent listens on localhost:19090, sweeps your LAN, talks Modbus, runs Lua drivers in a sandbox. Nothing else. No state on disk, no telemetry, no auto-update, no background services. Read every line in 30 minutes.

Architecture

Four parts. None of them lock you in.

The workbench runs entirely in your browser. Catalog, AI prompting, driver editor, live test results.

The agent is the open-source Go binary on your machine that the workbench talks to — over HTTP on localhost, or over NATS for headless deployments.

The API (Sourceful-hosted) proxies the Claude API key, handles GitHub OAuth, and signs short-lived NATS credentials. Open-source code; your AI usage is logged but never sold.

The driver library is a public GitHub repo — every accepted driver lands there under your name. Any EMS consumes it via the catalog API.
